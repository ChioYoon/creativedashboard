## 📊 핵심 변경 사항

### 1. ROAS 지표 추가 (매출 효율성 평가)
```
ROAS = 매출(Revenue) / 비용
- ROAS > 3.0: 핵심 소재 (예산 증액)
- ROAS 2.0~3.0: 우수 소재 (현상 유지)
- ROAS 1.0~2.0: 보통 소재 (모니터링)
- ROAS < 1.0: 개선 필요 (적자 상태)
```

### 2. Rank 기반 점수 계산 통일
```javascript
// 기존 (v1.4): 전환수만 비율 기반
전환수 점수 = (해당 소재 전환수 / 전체 합계) × 100

// 신규 (v1.5): 모든 지표 Rank 기반
1. 각 지표별 Rank 부여 (전환수↓, CPA↑, IPM↓, ROAS↓)
2. 점수 = (n - Rank + 1) / n × 100
3. Total Score = 가중 평균 (각 25%)

장점:
✓ 이상치 영향 최소화
✓ 공정한 비교 (모든 지표 0-100 스케일)
✓ 명확한 상대적 순위
```

### 3. 캠페인 복수 선택
```
사용법: Ctrl (Windows) 또는 Cmd (Mac) + 클릭
활용: 
- AB 테스트 비교 ("테스트_A" + "테스트_B")
- 지역별 통합 분석 ("KR" + "US" + "JP")
- 유사 캠페인 비교 ("신규_유저" + "재방문_유저")
```

### 4. 가중치 직접 입력
```
UI 구성:
[슬라이더] [입력 필드: 25] [25%]

특징:
- 슬라이더 변경 → 입력 필드 & 표시 업데이트
- 입력 필드 변경 → 슬라이더 & 표시 업데이트
- 0~100 범위 자동 제한
- 정확한 1% 단위 조정 가능
```

## 🚀 사용 가이드

### 기본 사용법
```
1. step1_integrated.html 열기
2. CSV 업로드 (매출/Revenue 컬럼 포함 권장)
3. 가중치 조정:
   - 슬라이더: 드래그로 조정
   - 입력 필드: 직접 숫자 입력 (예: 30)
4. 캠페인 선택: Ctrl + 클릭으로 복수 선택
5. "📊 점수 계산 실행" 클릭
6. 결과 확인: ROAS, Rank 기반 점수
```

### 가중치 전략 예시
```
전환 중심:     전환 40% / CPA 30% / IPM 15% / ROAS 15%
수익 중심:     전환 15% / CPA 15% / IPM 10% / ROAS 60%
균형 (기본):  전환 25% / CPA 25% / IPM 25% / ROAS 25%
효율성 중심:   전환 20% / CPA 40% / IPM 30% / ROAS 10%
```
