## 🔄 변경 상세

### 1. 점수 계산 공식

#### 변경 전 (v1.3)
```javascript
// 전환율 점수 계산
const maxConv = Math.max(...creatives.map(c => c.전환율));
c.전환율점수 = maxConv > 0 ? (c.전환율 / maxConv) * 100 : 0;

// Total Score
c.TotalScore = (
  c.전환율점수 * convWeight +
  c.CPA점수 * cpaWeight +
  c.IPM점수 * ipmWeight
);
```

#### 변경 후 (v1.4)
```javascript
// 전체 전환수 합계 계산
const totalConversions = creatives.reduce((sum, c) => sum + c.전환, 0);

// 전환수 점수 = (해당 소재 전환수 / 전체 전환수) × 100
c.전환수점수 = totalConversions > 0 ? (c.전환 / totalConversions) * 100 : 0;

// Total Score
c.TotalScore = (
  c.전환수점수 * convWeight +
  c.CPA점수 * cpaWeight +
  c.IPM점수 * ipmWeight
);
```

## 💡 개선 효과

### 1. 개념적 명확성
- **기존**: 전환율과 IPM이 사실상 동일 개념 (전환 / 노출수)
- **개선**: 전환수(절대값)와 IPM(상대값)으로 명확히 분리

### 2. 직관적 해석
- **기존**: "전환율이 최대 소재 대비 75%입니다" → 의미 파악 어려움
- **개선**: "전체 전환의 30%를 차지합니다" → 즉시 이해 가능

### 3. 실무 활용성
- **기존**: 효율성 위주 평가
- **개선**: 실제 비즈니스 기여도 직접 반영

### 4. 수학적 일관성
- **기존**: 전환율 점수 합계가 임의의 값
- **개선**: 전환수 점수 합계가 항상 100 (비율의 합)

## 📊 실무 활용 가이드

### 전환수 기반 점수 해석

| 전환수 점수 | 의미 | 액션 |
|-----------|-----|-----|
| 30점 이상 | 전체 전환의 30% 이상 기여 | 핵심 소재 - 예산 증액 고려 |
| 10~30점 | 전체 전환의 10~30% 기여 | 주력 소재 - 현상 유지 |
| 5~10점 | 전체 전환의 5~10% 기여 | 보조 소재 - 모니터링 |
| 5점 미만 | 전체 전환의 5% 미만 기여 | 저효율 소재 - 개선 필요 |

### 가중치 조정 전략

#### 전환수 중심 (신규 캠페인)
```
전환수: 60%
CPA: 25%
IPM: 15%
→ 실제 전환 창출 능력 우선
```

#### 균형 전략 (기본)
```
전환수: 40%
CPA: 35%
IPM: 25%
→ 기여도 + 효율성 + 잠재력 균형
```

#### 효율성 중심 (성숙 캠페인)
```
전환수: 20%
CPA: 50%
IPM: 30%
→ 비용 효율 최적화 우선
```
