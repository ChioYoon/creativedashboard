## 📋 개요

각 컨셉은 다른 톤앤매너와 레이아웃을 가지며, 텍스트와 데이터 가독성에 최적화되어 있습니다.

## 🎨 컴투스 브랜드 컬러 가이드

### Primary Color
- **레드/오렌지 계열**: `#E84855`, `#FF6B6B`, `#FF4757`
- 활기차고 역동적인 게임 회사의 이미지
- 주요 CTA 버튼, 강조 요소에 사용

### Secondary Color
- **다크 그레이/블랙**: `#2C3E50`, `#1A1A1A`
- 전문성과 신뢰성 표현
- 텍스트, 푸터 배경에 사용

### Neutral Color
- **라이트 그레이/화이트**: `#F8F9FA`, `#FAFAFA`, `#E9ECEF`
- 깔끔하고 읽기 편한 배경
- 카드, 섹션 배경에 사용

## 🔍 컨셉별 상세 분석

### 🎩 **컨셉 A: 프로페셔널 & 미니멀**

#### 시각적 특징
```css
헤더: 레드-오렌지 그라데이션 (#E84855 → #FF6B6B)
배경: 라이트 그레이 (#FAFAFA)
카드: 화이트 + 얇은 보더 + 그림자
버튼: 그라데이션 + 중간 그림자
```

#### 레이아웃
- **헤더**: 로고 + 버전 배지 (2단 구성)
- **히어로**: 중앙 정렬, 큰 타이틀 (42px)
- **기능 카드**: 3x2 그리드, 아이콘 + 제목 + 설명
- **통계**: 4칸 그리드, 숫자 + 라벨
- **푸터**: 중앙 정렬 텍스트

#### 장점 ✅
- 안정적이고 신뢰감 있는 디자인
- 텍스트 가독성 최우수 (큰 여백, 명확한 계층)
- 컴투스 기업 이미지에 가장 부합
- 마케팅 전문가에게 적합한 비즈니스 톤

#### 단점 ⚠️
- 다소 보수적인 느낌
- 시각적 임팩트가 약함
- 젊은 느낌이 부족

#### 추천 대상
✅ **임원진, 외부 발표용, 공식 문서**에 적합  
✅ 전문성과 신뢰성을 강조하고 싶을 때  
✅ 데이터와 텍스트를 명확하게 전달해야 할 때

---

### 🚀 **컨셉 B: 모던 & 다이나믹**

#### 시각적 특징
```css
헤더: 투명 블러 (backdrop-filter: blur(20px))
배경: 화이트 + 부유하는 도형 애니메이션
카드: 화이트 + 상단 라인 애니메이션
버튼: 큰 그림자 + 강한 호버 효과
통계: 그라데이션 배경 섹션 (레드-오렌지)
```

#### 레이아웃
- **헤더**: 블러 네비 + 로고 아이콘 (정사각형) + 링크
- **히어로**: 배지 + 큰 타이틀 (56px) + 2개 버튼
- **기능 카드**: 3x2 그리드, 아이콘 박스 + 제목 + 설명
- **통계**: 전체 너비 그라데이션 배경
- **푸터**: 3단 그리드 (About / Links / Contact)

#### 장점 ✅
- 현대적이고 세련된 디자인
- 애니메이션으로 생동감 표현
- 시각적 임팩트 강함
- Z세대, MZ세대에게 어필

#### 단점 ⚠️
- 애니메이션이 산만할 수 있음
- 데이터 분석 도구로는 과한 디자인
- 브라우저 성능에 따라 블러 효과가 다르게 보임

#### 추천 대상
✅ **팀 내부용, 젊은 팀원 대상**에 적합  
✅ 혁신적이고 트렌디한 이미지를 원할 때  
✅ 시각적 매력을 강조하고 싶을 때

---

### 📊 **컨셉 C: 클린 & 데이터 중심**

#### 시각적 특징
```css
헤더: 화이트 + 얇은 보더
배경: 라이트 그레이 (#F8F9FA)
카드: 보더 + 번호 라벨 (01, 02, 03...)
버튼: 단색 레드 + 섬세한 그림자
통계: 화이트 카드 + 보더
```

#### 레이아웃
- **헤더**: 미니멀 로고 (C 정사각형) + 링크 + 버전 태그
- **메인**: 레이블 + 타이틀 (48px) + 서브타이틀
- **기능 카드**: 2x3 그리드, 번호 + 제목 + 설명
- **통계**: 4칸 화이트 카드 그리드
- **정보 박스**: 그라데이션 CTA 섹션
- **푸터**: 3단 그리드 (대문자 제목)

#### 장점 ✅
- 가장 깔끔하고 정돈된 디자인
- 데이터 중심 레이아웃으로 정보 전달 최적화
- 번호 라벨로 명확한 순서 표시
- 로딩 속도 가장 빠름 (애니메이션 없음)

#### 단점 ⚠️
- 다소 밋밋한 느낌
- 시각적 흥미 요소 부족
- 너무 기능 중심적

#### 추천 대상
✅ **실무 분석용, 데일리 업무용**에 적합  
✅ 빠른 로딩과 명확한 정보 전달이 중요할 때  
✅ 장시간 사용해도 피로감이 적은 디자인 필요 시
