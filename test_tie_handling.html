🧪 동점자 처리 테스트 (v1.5.1)

📝 테스트 목적

문제: ROAS가 동일한 소재들이 다른 점수를 받는 문제

해결: 동점자에게 동일한 Rank 부여

예시:
기존 (잘못됨):
- 소재A: ROAS 2.0 → Rank 1 → 점수 100
- 소재B: ROAS 2.0 → Rank 2 → 점수 75 ❌ (동일한데 다른 점수)
- 소재C: ROAS 1.5 → Rank 3 → 점수 50

개선 (올바름):
- 소재A: ROAS 2.0 → Rank 1 → 점수 100
- 소재B: ROAS 2.0 → Rank 1 → 점수 100 ✅ (동일 Rank)
- 소재C: ROAS 1.5 → Rank 3 → 점수 50
