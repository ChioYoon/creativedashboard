## 🔄 변경 전후 비교

### 점수 계산 공식

#### 변경 전 (v1.3)
```
전환율 = (전환 / 노출수) × 100
전환율 점수 = (해당 소재 전환율 / 최대 전환율) × 100

문제점:
- 전환율과 IPM이 사실상 동일 개념
- 점수 합계가 일정하지 않음
- 실제 기여도를 직접 반영하지 못함
```

#### 변경 후 (v1.4) ✨
```
전환수 점수 = (해당 소재 전환수 / 전체 소재 전환수 합계) × 100

장점:
✓ IPM과 개념 중복 제거
✓ 점수 합계 항상 100
✓ 실제 전환 기여도 직접 반영
✓ 더 직관적인 해석 가능
```

### 샘플 계산 예시

**입력 데이터**:
```
소재A: 전환 500건 (전체의 50%)
소재B: 전환 300건 (전체의 30%)
소재C: 전환 150건 (전체의 15%)
소재D: 전환 50건 (전체의 5%)
전체: 1,000건
```

**v1.3 (전환율 기반)**:
```
소재A: 100.00점 (최대 전환율 대비)
소재B: 75.00점
소재C: 50.00점
소재D: 25.00점
→ 합계: 250.00점 (의미 없음)
```

**v1.4 (전환수 기반)** ✨:
```
소재A: 50.00점 (전체의 50%)
소재B: 30.00점 (전체의 30%)
소재C: 15.00점 (전체의 15%)
소재D: 5.00점 (전체의 5%)
→ 합계: 100.00점 (비율의 합)
```

## 💡 실무 활용 가이드

### 전환수 점수 해석

| 점수 | 의미 | 실무 액션 |
|-----|-----|---------|
| **30점 이상** | 전체 전환의 30% 이상 기여 | 🟢 핵심 소재 - 예산 증액 고려 |
| **10~30점** | 전체 전환의 10~30% 기여 | 🔵 주력 소재 - 현상 유지 |
| **5~10점** | 전체 전환의 5~10% 기여 | 🟡 보조 소재 - 모니터링 |
| **5점 미만** | 전체 전환의 5% 미만 기여 | 🔴 저효율 소재 - 개선 필요 |

### 가중치 조정 전략

#### 신규 캠페인 (전환 창출 중심)
```
전환수: 60%
CPA: 25%
IPM: 15%
→ 실제 전환 창출 능력 우선
```

#### 성장 캠페인 (균형)
```
전환수: 40% (기본값)
CPA: 35%
IPM: 25%
→ 기여도 + 효율성 + 잠재력 균형
```

#### 성숙 캠페인 (효율 최적화)
```
전환수: 20%
CPA: 50%
IPM: 30%
→ 비용 효율성 최우선
```

- 등급 배지: 최우수 / 우수 / 양호 / 보통 / 개선필요

## 📞 문제 해결

### Q: 점수 계산이 예상과 다릅니다
**A**: 전환수 점수는 전체 전환 중 해당 소재의 비율입니다.
- 예: 전환 100건 / 전체 1000건 = 10.0점

### Q: 점수 합계가 100이 안 됩니다
**A**: `test_conversion_score.html`을 열어 자동 테스트를 실행하세요.
- 5개 테스트 모두 통과해야 합니다.

### Q: 이전 분석 결과와 다릅니다
**A**: 정상입니다! v1.4는 전환율 대신 전환수를 사용합니다.
- 더 직관적이고 실무에 유용한 지표입니다.
